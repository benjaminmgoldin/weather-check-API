<h1>Weather Checker with API Call</h1>


<h2>Description</h2>
Project consists of a simple Python script to check the weather by entering a city. The script uses the free openweathermap API and requires a free API key which can be created at openweathermap.org. Currently, the API grants the temperature in kelvin. The scripts converts the temperature given in kelvin to fahrenheit.
<br />


<h2>Languages and Programs Used</h2>

- <b>Python</b> 
- <b>Microsoft Visual Studio 2022</b>

<h2>Environments Used </h2>

- <b>Windows 10</b> (21H2)
